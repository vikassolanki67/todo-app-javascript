# 📝 Modern Todo App

A modern and responsive Todo App built with **HTML**, **CSS**, and **Vanilla JavaScript**.

## ✨ Features

* ✅ Add new tasks
* ✅ Mark tasks as completed
* ✅ Delete tasks
* ✅ LocalStorage support
* ✅ Responsive Design
* ✅ Modern UI
* ✅ Custom Favicon

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* LocalStorage API

## 📂 Project Structure

```text
Todo-App/

│
├── index.html

```

## 🚀 Getting Started

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## 📱 Responsive

The project works on desktop, tablet, and mobile devices.
